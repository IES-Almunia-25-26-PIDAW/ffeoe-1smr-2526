## Objetivos

Al finalizar esta actividad el alumno será capaz de:

- Elaborar un inventario detallado del parque informático de la empresa.
- Identificar y documentar las características técnicas de los equipos.
- Detectar equipos con necesidades de mantenimiento o sustitución.
- Utilizar herramientas software de inventariado.

## Materiales necesarios

- Herramienta de inventariado: CPU-Z, Speccy, HWiNFO o equivalente.
- Hoja de cálculo (Excel, Calc) o GLPI / OCS Inventory si la empresa lo usa.
- Etiquetas o referencia de número de inventario de cada equipo.

## Desarrollo

### 1. Selección de la muestra

Selecciona un mínimo de **5 equipos** representativos del parque informático de la empresa
(o todos los disponibles si son menos). Incluye al menos un equipo de sobremesa, un portátil
y, si existe, un servidor de gama baja o NAS.

### 2. Inventario hardware

Para cada equipo, cumplimenta la siguiente ficha:

| Campo | Valor |
|---|---|
| Nº de inventario / etiqueta | |
| Tipo de equipo | Sobremesa / Portátil / Servidor / Otro |
| Marca y modelo | |
| Procesador (modelo y velocidad) | |
| RAM (capacidad y tipo) | |
| Almacenamiento (tipo, capacidad) | |
| Tarjeta gráfica | |
| Sistema operativo instalado | |
| Fecha de compra aproximada | |
| Estado general (Bueno / Regular / Malo) | |
| Observaciones | |

### 3. Inventario de periféricos

Para cada puesto de trabajo, anota los periféricos asociados: monitor (tamaño y resolución),
teclado, ratón, impresora, escáner, etc. Indica si son compartidos o individuales.

### 4. Análisis y propuesta

Tras el inventario, redacta un pequeño informe (media página) que incluya:

- Resumen del estado general del parque.
- Equipos que requieren mantenimiento inmediato.
- Equipos candidatos a sustitución (antiguos u obsoletos).
- Propuesta de actuación prioritaria.

## Documentación a entregar

- Tabla de inventario completa (todos los equipos seleccionados).
- Tabla de periféricos.
- Informe de análisis y propuesta de actuación.

## Evaluación

| Criterio | Peso | Descripción |
|---|---|---|
| Completitud del inventario | 40 % | Todos los campos cubiertos con datos reales |
| Precisión técnica | 30 % | Datos correctos contrastados con herramienta software |
| Informe de análisis | 30 % | Conclusiones coherentes y propuesta justificada |
