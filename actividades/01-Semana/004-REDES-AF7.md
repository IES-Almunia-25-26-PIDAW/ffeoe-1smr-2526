## Redes Locales — AF7

**Actividad Formativa 7 — Reconocimiento de la red**

Para esta actividad necesitarás:

- Un plano del C.E.I.P.
- Cuaderno o papel para hacer anotaciones

**Producto final:** Documentación de la red que contenga lo siguiente:

- **Doc 1 — Plano del CEIP** con la ubicación de todos los armarios identificados, la ubicación de todas las tomas en roseta y el recorrido del cableado.
- **Doc 2 — Para cada armario:** esquema con los elementos que contiene identificados y lista de los elementos con breve descripción.
- **Doc 3 — Esquema lógico** de conexión de los elementos de interconexión.
- **Doc 4 — Listado de VLANs** y rango de direcciones IP en cada una de ellas.
- **Doc 5 — Listado de conexiones.** Tabla con todos los puertos de los paneles de parcheo:

| Armario / Panel / Puerto | Etiqueta | Sala donde termina | Etiqueta en roseta | Verificada | Switch / Puerto | VLAN | Conexión / Velocidad | Elemento conectado |
| ------------------------ | -------- | ------------------ | ------------------ | ---------- | --------------- | ---- | -------------------- | ------------------ |
|                          |          |                    |                    |            |                 |      |                      |                    |

**Parte 1 — Localización de elementos en el plano**

En el plano del centro, toma nota de la ubicación de:

- El armario principal
- Los armarios de distribución, si los hay
- Cada una de las rosetas, indicando la etiqueta de las tomas que contiene
- Recorridos del cableado (habrá partes no visibles; intuye por dónde van)

**Parte 2 — Contenido del armario principal y los armarios de distribución**

De cada armario:

- Toma nota de los elementos que hay. Haz una lista indicando para cada uno qué función tiene.
- Haz un esquema gráfico de esos elementos dentro del armario.

**Parte 3 — Identificación de todos los puertos de paneles de parcheo y rosetas**

- Comprueba que todos los puertos de los paneles de parcheo tienen una etiqueta identificable y que sabes dónde termina (en qué roseta).
- Comprueba que todas las rosetas están bien identificadas y sabes dónde terminan (en qué armario, panel y puerto).
- Si algún puerto o toma no está identificado, investiga para intentar averiguarlo.
- Completa en la tabla las columnas: Armario / Panel / Puerto, Etiqueta, Sala donde termina, Etiqueta en roseta.
- Si es posible desconectar temporalmente lo que esté conectado, verifica la conexión con un tester y anota el resultado.

**Parte 4 — Esquema lógico de la red**

- Identifica todos los switches de la instalación y representa un esquema lógico de su conexión. Si algún switch está fuera de un armario, indícalo.
- Localiza en la documentación del armario las VLANs definidas e identifica a qué VLAN está asignado cada puerto.

**Parte 5 — Estado de conectividad de la red**

- Para cada puerto del panel de parcheo, comprueba si tiene un latiguillo hasta el switch correspondiente. Completa las columnas: Switch / Puerto, VLAN, Conexión / Velocidad.
- Para cada puerto del panel de parcheo, intenta identificar qué elemento hay conectado e indícalo en la tabla.

**Entrega:**

- Para el profesorado: Doc 7.1 a Doc 7.5
- Para el CEIP:
  - Documento PDF resultado de combinar Doc 7.1 a Doc 7.5
  - Si lo solicitan, el resto de archivos en formato editable
  - La tabla Doc 7.5 como hoja de cálculo (preferiblemente en Google Docs compartido)
